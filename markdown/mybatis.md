# mybatis 框架
1. 什么是框架
    - 框架是我们软件开发中的一套解决方案，不同的框架解决的是不同的问题。
    - 使用框架的好处：
    > 框架封装了许多的细节，使开发者可以使用极简的方式实现功能，大大提高开发效率

2. 三层架构
   - 表现层：用于展示数据
   - 业务层：处理业务需求
   - 持久层：负责和数据库进行交互

3. 持久层技术解决方案
   - JDBC 技术：
     - Connectiong
     - PrepareStatement
     - ResultSet
   - Spring 的 JdbcTemplate：
     - Spring 对 jdbc 的简单封装
   - Apache 的 DBUtils：
     - 也是对 jdbc 的简单封装
